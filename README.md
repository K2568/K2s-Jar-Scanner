# K2s-Jar-Scanner
Gives you a list of embedded mods within mods and also scans for MCreator
This is completely safe to use, it only ever calls read functions and never write. You can read through the file if you want to.
Report issues on the isses page

## Instructions:
Don't run this in your downloads folder or it will scan every jar you have in there and could lag.
Make a folder somewhere and put this file and any jar files of mods you want to use in the same folder as it.
Run the python program and it will give you an output.

### Exmaple usage:

effortlessbuilding  -  1.21.1  -  3.10  -  all.jar
      flywheel  -  neoforge  -  1.21.1  -  1.0.2.jar
      Ponder  -  NeoForge  -  1.21.1  -  1.0.46.jar

protection_pixel  -  2.0.2  -  forge  -  1.20.1.jar
      No embedded mods
      MCREATOR DETECTED
